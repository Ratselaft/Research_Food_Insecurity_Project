NLP Findings: Topics, Keywords, and Theme-Variable Mapping

TF-IDF identifies the most distinctive terms across the strictly aligned corpus. NMF (k=7) groups those terms into interpretable topics. Each topic is then mapped to an empirical proxy variable used in the regression models.
